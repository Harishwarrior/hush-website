# Hush — Silence the Noise, Keep What Matters 🔔

Hush is a privacy-first notification management application for Android that helps you filter out the noise and focus on what truly matters.

## Features

- **Smart Categorization**: Automatically filters and categorizes your notifications on-device.
- **Privacy-First**: No data collection, no accounts, and no tracking. All notification processing happens locally.
- **Interactive Mockup**: A custom-built OnePlus 6T device frame showcasing the application's core workflow.
- **Modern Design**: Built with high-performance animations and a clean, accessible interface.

## Privacy

Privacy is the foundation of Hush. We do not collect, store, or share any personal data or notification content. Read our full [Privacy Policy](/privacy) for more details.
